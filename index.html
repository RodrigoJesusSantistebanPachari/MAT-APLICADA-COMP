Bases ortonormales y proyección

Una base ortonormal es una base ortogonal, en la cual la norma de cada elemento que la compone es unitaria.
Por lo tanto, para obtener una base ortonormal es necesario obtener una base ortogonal.
Para poder obtener una base ortogonal es necesario realizar el proceso de Gram-Schmidt
. Este proceso se resumen en que para obtener una base ortogonal k de un vector k, se tiene que seguir el siguiente procedimiento:
$$u_k = v_k - \sum_{j=1}^{k-1} Proy_{v_k}(u_k) $$
La proyección de un vector sobre otro se obtiene de la siguiente forma:
$$Proy_{v_k}(u_k) = \frac{<\v_k, u_j>}{<\u_j, u_j>} u_j$$
Por lo tanto, para obtener una base usando el proceso de ortogonalización, se resumen de la siguiente forma:
$${u_k = v_k - \sum_{j=1}^{k-1}\frac{<\v_k, u_j>}{<\u_j, u_j>} u_j}$$

Bases ortonormales

Ahora que se ha entendido las bases ortogonales, para obtener una base ortonormal a partir de una base ortogonal, se tiene que normalizar. Esto se hace de la siguiente manera:
$$w_1 = \frac{u_1}{|u_1|}$$
La base $w_1$ representa una base ortonormal, su obtención es la división entre la base ortogonal $u_1$ y el módulo de esta base ortogonal.
